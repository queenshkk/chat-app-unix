# UNIX Chat Application (C / Linux)
Application de messagerie réalisée en langage C dans le cadre d'un projet UNIX.

## Aperçu
Interface permettant :

- La connexion d'un utilisateur avec nom et mot de passe
- L'envoi de messages
- L'affichage des personnes connectées
- La consultation des informations des utilisateurs

## Technologies utilisées
- Langage C
- Programmation UNIX

## Compilation
Compiler le projet avec : make

## Lancer le programme
Lancer ./Client
